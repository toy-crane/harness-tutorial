### Components

| # | Name | Type | Detail |
|---|------|------|--------|
| ① | 앱 로고/제목 | Text | "Kanban Board" |
| ② | 내보내기/가져오기 버튼 | Button | Export → JSON 다운로드, Import → 파일 선택 후 덮어쓰기 |
| ③ | 다크모드 토글 | Toggle | 라이트/다크 전환, localStorage 저장 |
| ④ | 검색 입력 | Input | 카드 제목 필터링, 실시간 반영 |
| ⑤ | 우선순위 필터 | Select | All / High / Medium / Low |
| ⑥ | 태그 필터 | Select | All / 등록된 태그 목록 |
| ⑦ | 칼럼 헤더 | Text | 칼럼명 + 카드 수 |
| ⑧ | 카드 | Card | 클릭 시 편집 모달 열림, 드래그&드롭 가능 |
| ⑨ | 태그 뱃지 | Badge | 카드에 설정된 태그 표시 |
| ⑩ | 마감일 | Text | 날짜 표시 |
| ⑪ | 마감일 경고 | Indicator | 마감일 < 오늘 → 빨간 테두리 |
| ⑫ | 서브태스크 진행률 | Progress | 완료/전체 (예: 2/5) |

### Interactions

- [②] Export 클릭 → JSON 파일 다운로드
- [②] Import 클릭 → 파일 선택 → 기존 데이터 덮어쓰기
- [③] 토글 클릭 → 테마 전환 + localStorage 저장
- [④] 텍스트 입력 → 제목에 매칭되는 카드만 표시
- [⑤] 우선순위 선택 → 해당 우선순위 카드만 표시
- [⑥] 태그 선택 → 해당 태그 포함 카드만 표시
- [⑧] 카드 클릭 → 편집 모달 열림
- [⑧] 카드 드래그 → 칼럼 간 이동 또는 칼럼 내 순서 변경
- [⑧] 🗑 클릭 → 삭제 확인 대화상자
- [+] Add a card → 제목 입력 후 Todo 칼럼에 카드 추가

### Components

| # | Name | Type | Detail |
|---|------|------|--------|
| ① | 모달 헤더 | Text + Button | 카드 제목 표시 + ✕ 닫기 버튼 |
| ② | 제목 입력 | Input | 카드 제목 수정 |
| ③ | 우선순위 셀렉트 | Select | High / Medium / Low (색상 구분) |
| ④ | 마감일 피커 | Date Input | 날짜 선택 |
| ⑤ | 태그 목록 | Badge + Button | 기존 태그 ✕ 제거 + 새 태그 추가 |
| ⑥ | 서브태스크 체크리스트 | Checkbox + Button | 체크 토글 + 🗑 삭제 + 새 항목 추가 |

### Interactions

- [①] ✕ 클릭 → 모달 닫기 (변경 취소)
- [②] 제목 수정 후 Save → 보드에 반영
- [③] 우선순위 변경 → 카드 색상 변경
- [④] 날짜 선택 → 마감일 설정 (경과 시 빨간 테두리)
- [⑤] + Add 클릭 → 태그 입력 후 추가, ✕ 클릭 → 태그 제거
- [⑥] 체크박스 토글 → 진행률 업데이트, 🗑 클릭 → 서브태스크 삭제
- [⑦] Save → 모든 변경 저장 + 모달 닫기
