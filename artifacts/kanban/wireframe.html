프로젝트 셋업

하
Notes

칸반 보드의 기본 화면. 3개 고정 칼럼(To Do, In Progress, Done)에 카드를 배치한다.

  • 카드 생성: To Do 칼럼 하단 input에 제목 입력 후 + 버튼 클릭. 빈 제목 시 "제목을 입력해주세요" 에러. 기본 우선순위 "중"
  • 드래그&드롭: 카드를 다른 칼럼으로 드래그하여 이동. 같은 칼럼 내 순서 변경도 가능
  • 검색: 검색창에 텍스트 입력 시 제목에 포함된 카드만 실시간 필터링. 검색어 삭제 시 전체 복원
  • 필터: 우선순위/태그 드롭다운 선택 시 조건에 맞는 카드만 표시. 복수 필터는 AND 조합
  • 다크모드: 토글 클릭 시 라이트/다크 전환. 설정은 localStorage에 영속화
  • 내보내기: 클릭 시 전체 보드 데이터가 JSON 파일로 다운로드
  • 데이터 영속화: 모든 변경(추가/수정/삭제/이동)은 localStorage에 자동 저장. 새로고침 시 복원

Notes

카드 클릭 시 인라인 편집 모드로 전환되는 화면. 편집 중인 카드는 굵은 테두리로 강조된다.

  • 인라인 편집: 카드 클릭 → 제목, 설명, 우선순위, 태그, 마감일 필드가 편집 가능한 form으로 확장
  • 제목 변경: "회의록 작성" → "주간 회의록" 등. 빈 값 시도 시 "제목을 입력해주세요" 에러 표시
  • 우선순위 변경: 드롭다운에서 상/중/하 선택
  • 태그 자동완성: "버" 입력 시 기존 태그 "버그"가 자동완성 목록에 표시됨. 태그 옆 ✕ 클릭으로 제거
  • 편집 외부 클릭 시 편집 모드 종료 (변경사항 자동 저장)

카드 삭제 ✕
⚠

정말 삭제하시겠습니까?

"회의록 작성" 카드가 영구적으로 삭제됩니다.

Notes

카드의 🗑 버튼 클릭 시 표시되는 삭제 확인 다이얼로그.

  • "삭제" 클릭 → 카드가 목록에서 제거되고 localStorage에 반영
  • "취소" 또는 ✕ 클릭 → 다이얼로그 닫히고 카드 유지
  • 삭제할 카드명이 다이얼로그 본문에 표시됨
