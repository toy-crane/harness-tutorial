중 개발
Notes

칸반 보드의 기본 상태. 3개 고정 칼럼(To Do, In Progress, Done)에 카드가 배치되어 있다.

  • 카드 생성: 각 칼럼 하단 입력 필드에 제목 입력 후 + 버튼 클릭. 빈 제목 시 "제목을 입력해주세요" 에러 표시. 기본 우선순위 "중"
  • 드래그&드롭: 카드를 다른 칼럼으로 드래그하여 이동. 같은 칼럼 내 순서 변경도 가능
  • 검색: 상단 검색창에 텍스트 입력 시 제목에 포함된 카드만 표시. 검색어 삭제 시 전체 표시
  • 필터: 우선순위/태그 드롭다운으로 조건 필터링. 복수 조건은 AND 조합. 필터 해제 시 전체 표시
  • 다크모드: 토글 클릭 시 라이트/다크 전환. 설정은 새로고침 후에도 유지 (localStorage)
  • 내보내기: 클릭 시 전체 보드 데이터가 JSON 파일로 다운로드
  • 가져오기: 클릭 시 파일 선택 → 확인 다이얼로그 → 데이터 교체 (Screen 4 참조)
  • 영속화: 모든 변경(추가/수정/삭제/이동)은 localStorage에 자동 저장, 새로고침 시 복원

중 개발
Notes

카드를 클릭하면 해당 카드가 인라인 편집 모드로 전환된다. 편집 중인 카드는 두꺼운 테두리로 구분.

  • 편집 필드: 제목, 설명, 우선순위(상/중/하), 태그, 마감일을 모두 인라인으로 편집
  • 제목 변경: "회의록 작성" → "주간 회의록" 등 즉시 반영. 빈 값 시 "제목을 입력해주세요" 에러
  • 우선순위 변경: 드롭다운으로 상/중/하 선택
  • 태그 자동완성: 태그 입력 시 기존 태그 목록에서 자동완성. "버" 입력 → "버그" 표시
  • 태그 제거: 태그의 ✕ 클릭으로 해당 태그 제거
  • 카드 외부 클릭 시 편집 모드 종료

삭제 확인 (scenario: 3)

카드 삭제 ✕

정말 삭제하시겠습니까?

삭제된 카드는 복구할 수 없습니다.

Notes

카드의 🗑 버튼 클릭 시 표시되는 확인 다이얼로그.

  • "삭제" 클릭 → 카드가 목록에서 제거되고 다이얼로그 닫힘
  • "취소" 클릭 또는 ✕ 클릭 → 카드가 그대로 유지되고 다이얼로그 닫힘

Notes

"↓ 가져오기" 버튼 클릭 시 표시되는 다이얼로그. 파일 선택과 확인을 처리한다.

  • 파일 선택 후 "가져오기" 클릭 → 기존 데이터가 선택한 파일의 데이터로 전체 교체
  • "취소" 클릭 → 기존 데이터 유지, 다이얼로그 닫힘
  • 잘못된 형식의 파일 선택 시 → "올바른 형식의 파일이 아닙니다" 에러 Alert 표시
  • 에러 Alert는 정상 파일 선택 시에는 숨김 상태
