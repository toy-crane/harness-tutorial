# 와이어프레임 예시

kanban-todo 기반 예시입니다. 아래 패턴을 모두 시연합니다:
- 시각적 버튼/인풋 (시맨틱 레이블 없음)
- 실제 한국어 데이터
- `(● High)` 스타일 배지
- 점선 빈 상태
- `①②③` 번호 주석 → Component Inventory 연결
- 멀티 칼럼 레이아웃

## Component Inventory 예시 (Desktop 기준)

| # | Component | Type | Content/State |
|---|-----------|------|---------------|
| ① | 새 할일 버튼 | Button | label: "+ 새 할일", variant: default |
| ② | 아바타 | Avatar | 사용자 이니셜 "JD" |
| ③ | 할 일 카드 | Card | title + badge + 날짜, draggable |
| ④ | 우선순위 배지 | Badge | "High" (● dot), variant: destructive |
| ⑤ | 빈 상태 추가 | Empty state | 점선 테두리, 클릭 시 새 카드 생성 |
